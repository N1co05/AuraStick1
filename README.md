# 🌸 Aura Stickers — Rediseño Profesional

¡Bienvenido al repositorio de **Aura Stickers**! Este es un sitio de e-commerce moderno, minimalista y de alto impacto visual, diseñado para ofrecer una experiencia premium a tus clientes.

## 🚀 Características Principales

- **Diseño de Clase Internacional**: Sistema de diseño basado en Glassmorphism, tipografía elegante y una paleta de colores curada.
- **Catálogo Dinámico**: Carga de productos en tiempo real directamente desde **Firebase Firestore**.
- **Panel de Administración**: Gestión completa de inventario con subida directa de imágenes a **Firebase Storage**.
- **Checkout por WhatsApp**: Carrito de compras integrado que genera pedidos automáticos listos para enviar por mensaje.
- **Optimizado para Móviles**: Experiencia fluida en cualquier dispositivo.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: HTML5 Semántico, CSS3 (Custom Variables, Animaciones Avanzadas), Vanilla JavaScript (ES6+).
- **Backend & DB**: Firebase (Firestore, Storage, Web SDK v10).
- **Hosting Sugerido**: GitHub Pages, Vercel o Netlify.

## 📦 Estructura del Proyecto

```text
pagina/
├── index.html          # Landing Page
├── catalogo.html       # Tienda y Carrito
├── admin.html          # Panel de Gestión
└── src/
    ├── assets/         # Imágenes y Logos
    ├── pages/          # Lógica JavaScript por página
    ├── services/       # Configuración de Firebase
    └── styles/         # Sistema de diseño global.css
```

## 🌐 Despliegue en GitHub Pages

Este proyecto está **100% adaptado para GitHub Pages**. No requiere configuración adicional.

1. Subí todos los archivos a un repositorio de GitHub.
2. Ir a **Settings** -> **Pages**.
3. Seleccioná el branch `main` y la carpeta `/ (root)`.
4. ¡Listo! Tu tienda estará online en segundos.
